# Automated EDA Tool (Interactive Data Analysis App)

An interactive web-based tool that performs automated exploratory data analysis (EDA) on any dataset.  
Users can upload a CSV file and instantly generate insights, visualisations, and data summaries without writing code.

## Features

- Upload any CSV dataset
- Automatic dataset overview (rows, columns, data types)
- Missing values analysis
- Distribution plots for numerical features
- Categorical feature breakdown
- Correlation heatmap
- Auto-generated insights (key patterns in data)

---

## Project Overview

Exploratory Data Analysis (EDA) is a crucial step in any data science workflow but it can be time-consuming and repetitive.  

This project automates the EDA process by building an interactive tool that:
- Quickly analyses datasets
- Identifies patterns and anomalies
- Helps users understand their data before modelling

The goal is to simulate how a data analyst would approach a new dataset but in an automated, reusable system.
